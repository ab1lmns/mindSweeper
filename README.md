# 💣 MindSweeper

**MindSweeper** — это соревновательная платформа, превращающая классического "Сапёра" в киберспорт. Вдохновлено подходом Chess.com: здесь есть система рейтинга (ELO), глобальные таблицы лидеров, ежедневные турниры и умные подсказки.

![MindSweeper Dashboard](https://media.discordapp.net/attachments/1083424683002634260/1179424683002634260/mindsweeper_preview.png) *(Иллюстративное превью)*

## 🌟 Главные фичи (Что уже работает)

*   **🏆 Ранговый режим (Ranked ELO)**
    *   Сражайся за рейтинг! Успешное разминирование повышает твой ELO, подрыв — снижает. Система автоматически пересчитывает рейтинг на базе твоего времени и сложности.
*   **🌍 Глобальный Лидерборд**
    *   Соревнуйся с игроками со всего мира. Задавай себе уникальный никнейм и пробивайся в ТОП-100 лучших сапёров.
*   **📅 Daily Challenge (Испытание дня)**
    *   Каждый день генерируется **одно уникальное поле**, одинаковое абсолютно для всех. У тебя есть только одна попытка, чтобы показать лучшее время и попасть в дневной топ.
*   **🧠 AI Coach (Умные вероятности)**
    *   Своеобразный аналог Stockfish из шахмат. Режим "сканера" показывает в реальном времени математическую вероятность нахождения мины под каждой закрытой клеткой.
*   **🔐 Профили и Авторизация**
    *   Вход через Email или Google-аккаунт. Вся статистика (винрейт, лучшее время на разных сложностях, количество игр) надежно сохраняется в облачной базе данных.
*   **🎨 Премиальный дизайн**
    *   Стиль **Glassmorphism** с неоновыми акцентами, плавными анимациями (Framer Motion) и продуманным UX/UI. Поддержка светлой и темной темы.

## 🛠 Технологический стек

*   **Frontend:** React 18, TypeScript, Vite
*   **Styling:** TailwindCSS, Lucide React (иконки)
*   **Animations:** Framer Motion
*   **Backend & Auth:** Supabase (PostgreSQL) — настроены RLS-политики, триггеры автосоздания профилей и Google OAuth.

## 🚧 План развития (Что не успел добавить)

Проект был собран в рекордные сроки как рабочий MVP. Вот список фич, которые запланированы на следующие апдейты, но пока не вошли в релиз:

1.  **💳 Оплата PRO-аккаунтов**
    *   В интерфейсе уже есть элементы PRO-подписки (которая давала бы бесконечный доступ к AI-подсказкам и кастомным скинам), но **платежный шлюз (Stripe/YooKassa) пока не прикручен**. 
2.  **⚔️ Мультиплеер (Live Battles)**
    *   В планах сделать режим 1v1, где два игрока одновременно решают одно и то же поле. Кто первый — тот забирает рейтинг.
3.  **🏅 Полная система ачивок**
    *   Таблицы в БД для достижений (Achievements) уже созданы, но выдача наград за специфические действия (например, "пройти хард за 2 минуты") пока находится в разработке.

## 🚀 Как запустить локально

1. Склонируй репозиторий.
2. Установи зависимости: `npm install`
3. Создай файл `.env` в корне проекта и добавь ключи от своего Supabase проекта:
   ```env
   VITE_SUPABASE_URL=твоя_ссылка
   VITE_SUPABASE_ANON_KEY=твой_ключ
   ```
4. Выполни в Supabase SQL Editor скрипт из папки `supabase/schema.sql` для создания базы.
5. Запусти проект: `npm run dev`
